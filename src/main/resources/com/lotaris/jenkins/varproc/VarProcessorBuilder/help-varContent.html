Define the variables you want to add to the job: 

	myAdditionalVariable=rnd(len:"12")
	mySecondAdditionalVariable=sha(strength:"256",str:"abcdef")

Available statements

Argument prefixed with a star are optional parameters.

Example Meaning
rnd(len:"12")
rnd(len:"12",range:"abcdef",prefix:"pref-",suffix:"-suff")

Generate a random string based on the args passed

len: Define the length of the string
*range: Define the range of values that will be used to generate the string, by default alapha-numeric values are used.
*prefix: Add a prefix to the string generated
*suffix: Add a suffix to the string generated
sha(strength:256,str:"stringToSha")

Generate a SHA of the given string

strength: The name of the SHA algorithm (1, 256, 512)
str: The string to SHA

Additional

If a string contain a valid Jenkins variable, it will be resolved as usual.
